# GourmetSearch
YahooのAPIを利用したグルメ検索アプリです。

## 機能
- ワード検索、現在地検索機能
- お気に入り保存
- 写真撮影 or カメラロールからお店毎に写真を追加できる
- 電話番号をタップで電話をかけられる
- 住所をタップで地図詳細画面へ
- 現在地ボタンで現在地とお店の住所を同時表示
- twitter, line, facebookへのシェア機能
